# Task 11

## Overview
This task implements an e-commerce style interface with product display, shopping cart, and payment pages. It includes a main product display page, a shopping cart page, and a payment page, styled with `style.css`.

## Files
- `index.html` - Main entry page for the task.
- `product-display.html` - Shows the available products and product details.
- `shopping-cart.html` - Displays the items added to the cart.
- `payment.html` - Contains the checkout/payment interface.
- `style.css` - Styling for all task pages.

## How to Use
1. Open `index.html` in a web browser.
2. Navigate to the product display section to view items.
3. Add products to the shopping cart.
4. Proceed to `shopping-cart.html` to review selected items.
5. Continue to `payment.html` to simulate checkout.

## Notes
- The pages are designed for static preview and demonstration.
- Additional interactivity may require JavaScript if not already included in the HTML files.
